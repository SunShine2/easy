$.EasyTouch Class

Extends $.Base
Defined in: js\core.js:260
Module: EasyTouch-Page
Parent Module: EasyTouch

EasyTouch，为移动设备设计的UI框架。使用方法：用$.EasyTouch.extend方法扩展一个App类，然后初始化。extend的具体说明，请看$.EasyTouch.extend；参数pages的说明请看$.EasyTouch.Page：

var App = $.EasyTouch.extend({
    id: xxx,
    container: xxx,
    pages: {
       ...
    },
    apps: {
       ...
    },
    events: {
       ...
    }
});

new App();

extend

(
  • property
)
Function static

Provided by the EasyTouch module.

Defined in js\core.js:1101

通过该方法来继承并扩展一个$.EasyTouch

var App = $.EasyTouch.extend({
    id: 'market',
    container: '#app',
    ifShowLoading: false,
    defaultAnimation: 'slideRightIn',
    debug: true,
    pages: {},
    apps: {
       'hongbao': 'http://cloudappfile.aliapp.com/prod/app_4/7184_b6b/hongbao/'
    },
    events: {
        'tap header': 'tapHeader',
        'tap .back': this.app.pageBack
    },
    //this function will run when creating instance
    init: function(data){},
    //for YunOS resume event
    resume: function(){},
    //for YunOS suspend event
    pause: function(){}
});
new App();

Parameters:

  • property Object

    需要扩展的属性列表

    • id String

      应用ID，用于EasyTouch内部识别

    • container Node | HTMLElement | String

      应用容器，默认为body

    • ifShowLoading Boolean

      页面加载过程是否显示加载中的提示，默认为true

    • defaultAnimation String

      默认的页面切换效果，默认为undefined

    • debug Boolean

      debug模式开关，默认为false

    • pages Object

      页面配置： 如果是一个Object，EasyTouch会调用$.EasyTouch.Page.extend，将这个Object作为prototype属性扩展出一个Page，具体请看$.EasyTouch.Page.extend； 如果是一个$.EasyTouch.Page.extend产生的Class，则将直接使用该类作为一个Page；

    • apps Object

      子应用配置：key为应用的id，value为应用的访问地址，子应用将作为一个iframe的形式存在

Returns:

Function: the new Class extended from $.EasyTouch

Parameters:

  • id String

    page id

  • data Object

    data for next page

  • anim String

    animation name, contain:

    • slideRightIn
    • slideLeftIn
    • slideTopIn
    • slideBottomIn
    • fadeIn
    • dissolveIn
    • popIn 4.0+
    • flipLeftIn 4.0+
    • flipRightIn 4.0+
    • swapLeftIn 4.0+
    • swapRightIn 4.0+
    • cubeLeftIn 4.0+
    • cubeRightIn 4.0+
    • flowLeftIn 4.0+
    • flowRightIn 4.0+
    • turnIn 4.0+

    @import "easytouch/less/anim.less";

//The same, when you include slideLeftOut, slideRightIn is ready too.
//All the animation function has two options: "duration" and "function".
.easytouch > .anim-slideRightIn(.6s, ease-in);
  • ifPushToHistory Boolean

    if push the page to history, default: true
