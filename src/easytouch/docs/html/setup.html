只加载用到的动画样式

在应用的less文件中加载anim.less文件，使用less的mix方法调用用到的动画

/*加载import less文件，路径需要修改*/
@import "/easytouch/less/anim.less";

/*slideLeftIn、slideLeftOut、slideRightIn、slideRightOut对应的mix方法*/
.easytouch > .anim-slide();
